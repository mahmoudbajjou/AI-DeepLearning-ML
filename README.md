# AI-DeepLearning-ML
A comprehensive collection of academic assignments and projects covering the fundamentals of Machine Learning and the architectures of Deep Learning.

## 📂 Repository Structure
The project is organized into two main pillars of Artificial Intelligence:

##  Machine Learning
This section focuses on classical statistical learning methods and optimization.

- SGD (Stochastic Gradient Descent): Implementation of optimization algorithms for large-scale learning.

- Logistic Regression: Classification tasks using probabilistic linear modeling.

- Avazu: A practical application focusing on Click-Through Rate (CTR) prediction using the Avazu dataset.

##  Deep Learning
Implementations of various neural network architectures using modern frameworks.

- MLP (Multi-Layer Perceptron): The foundation of feed-forward neural networks.

- CNN (Convolutional Neural Networks): Specialized layers for image processing and spatial feature extraction.

- RNN (Recurrent Neural Networks): Sequential data processing.

- LSTM (Long Short-Term Memory): Advanced recurrent units designed to handle the vanishing gradient problem in long sequences.

##  Tech Stack
- Language: Python

- Libraries: * NumPy & Pandas (Data manipulation)

- Matplotlib & Seaborn (Visualization)

- Scikit-Learn (Classical ML)

- PyTorch / TensorFlow (Deep Learning frameworks)
